# Lotes de Pruebas - Ejercicio 5

## Cómo ejecutar

### Bash
```bash
bash ./lote_pruebas.sh
```

O si tiene permisos de ejecución:
```bash
./lote_pruebas.sh
```

### PowerShell
```bash
pwsh ./lote_pruebas.ps1
```

**Importante:** En Linux/Mac, PowerShell debe ejecutarse explícitamente con `pwsh`, no con bash.

---

## Contenido de los lotes

Ambos archivos contienen ejemplos de comandos para probar los scripts `rickandmorty.sh` y `rickandmorty.ps1`, cubriendo:

- Ayuda y validación
- Búsqueda por ID
- Búsqueda por nombre
- Búsqueda combinada
- Validación de errores
- Gestión de caché
- Limpieza de caché
